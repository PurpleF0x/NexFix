# 🤖 NEX — Nexa OS Assistant (Android 16 Edition)

O **Nex** é um assistente proativo de nova geração, inspirado no protocolo JARVIS, desenvolvido especificamente para o ecossistema Android 16.

## 👤 Arquiteto & Criador
- **Martim** (@Purpl3F0x)

## 🚀 Funcionalidades Atuais
- **Voz de Fundo:** Ativação multigatilho ("Ei Nex", "Olá") em segundo plano.
- **Consciência de Hardware:** Monitorização térmica e de bateria com alertas proativos.
- **Controlo de Sistema:** Gestão de luz, volume, brilho, Wi-Fi e chamadas.
- **Auto-Conhecimento:** O Nex consulta os commits deste repositório para saber as suas próprias atualizações.
- **Visão de Ecrã:** Integração com serviços de acessibilidade para "ver" o conteúdo do telemóvel.

## 🧠 Tecnologia
- **Android:** Kotlin, Foreground Services, Speech API, Jetpack Compose.
- **Cérebro (Backend):** PHP (Render), Integração com API Groq (Llama-3.3-70b-versatile).
- **Memória:** Persistência via JSON com lógica de deduplicação.
